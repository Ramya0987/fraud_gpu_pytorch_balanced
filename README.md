# fraud_gpu_pytorch_balanced
# Financial Fraud Detection with GPU Acceleration

This project uses a GPU-accelerated deep learning model to detect financial fraud in real-time. Implemented in PyTorch with CUDA backend, the model classifies over 10,000 transactions with high accuracy and efficiency.

## 🚀 Tech Stack
- Python, PyTorch, CUDA
- Google Colab (Tesla T4 GPU)
- Scikit-learn, NumPy, Pandas

## 📊 Results
- **F1 Score:** 0.92  
- **Precision:** 0.89  
- **Recall:** 0.95  
- **3x speed improvement** vs CPU-only model

## 📁 Files
- `fraud_gpu_pytorch_balanced.ipynb`: Final training notebook
- `Financial_Fraud_Detection_GPU_Summary.pdf`: 1-page project summary
